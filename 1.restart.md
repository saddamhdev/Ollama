তুমি যেহেতু **Ollama** Ubuntu/WSL এ install করেছ, restart করার কয়েকটা উপায় আছে। নিচে সহজভাবে দিলাম।

---

# 1️⃣ Ollama Service Restart (Recommended)

কারণ install করার সময় এটি **systemd service** হিসেবে run হচ্ছে।

Restart command:

```bash
sudo systemctl restart ollama
```

Status check:

```bash
systemctl status ollama
```

Expected output:

```
Active: active (running)
```

---

# 2️⃣ Stop + Start Manually

Stop:

```bash
sudo systemctl stop ollama
```

Start:

```bash
sudo systemctl start ollama
```

---

# 3️⃣ Quick Restart (Terminal Way)

Sometimes developers simply do:

```bash
pkill ollama
ollama serve
```

---

# 4️⃣ Check API After Restart

Open:

```
http://127.0.0.1:11434
```

Or test:

```bash
curl http://localhost:11434
```

---

# 5️⃣ Test Model After Restart

```bash
ollama run deepseek-coder
```

Example:

```
write spring boot service example
```

---

# ⚠️ Important (WSL users)

তোমার prompt দেখাচ্ছে:

```
saddamnvn@DESKTOP
```

মানে তুমি সম্ভবত **Windows + WSL Ubuntu** ব্যবহার করছ।

WSL restart করলে Ollama service বন্ধ হয়ে যেতে পারে। তখন run করো:

```bash
sudo systemctl start ollama
```
