# 3. Install Ollama

Ollama local AI server run করে।

Install command:

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

Verify installation:

```bash
ollama --version
```

Example output:

```
ollama version 0.17.7
```

---

# 4. Start Ollama Server

Normally install করার সময় server start হয়ে যায়।

Check server:

```
http://127.0.0.1:11434
```

Manual start:

```bash
ollama serve
```

# 5. Download Coding Model

Recommended coding model:

```
deepseek-coder
```

Download:

```bash
ollama pull deepseek-coder
```

```
~700MB – 7GB
```

# 6. Run Model

Run interactive session:

```bash
ollama run deepseek-coder
```

Example prompt:

```
write a spring boot rest controller
```

Example output:

```java
@RestController
@RequestMapping("/users")
public class UserController {

    @GetMapping
    public List<User> getUsers() {
        return List.of(new User(1,"Saddam"));
    }
}
```

Exit:

```
/bye
```

# 8. Ollama API Usage

Ollama exposes a local REST API:

```
http://localhost:11434
```

Example curl request:

```bash
curl http://localhost:11434/api/generate -d '{
  "model": "deepseek-coder",
  "prompt": "write java hello world"
}'
```

Response:

```
public class HelloWorld {
    public static void main(String[] args){
        System.out.println("Hello World");
    }
}
```

---

# 10. Configure Continue

Config file location:

```
~/.continue/config.json
```

Example configuration:

```json
{
  "models": [
    {
      "title": "DeepSeek Local",
      "provider": "ollama",
      "model": "deepseek-coder"
    }
  ]
}
```

Restart IDE after saving.
