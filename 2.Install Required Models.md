# 3. Ollama Server

Ollama local AI server run করে।

API address:

```
http://localhost:11434
```

Check server:

```bash
curl http://localhost:11434
```

Restart service:

```bash
sudo systemctl restart ollama
```

Check status:

```bash
systemctl status ollama
```
